niceScroll 滚动条

  1. 引入：nicescroll.js 下载地址 https://github.com/inuyaksa/jquery.nicescroll/releases
  2. 初始化：必须在 $(document).ready 中进行。 
    $("html").niceScroll(options);
  3. 参数配置：options 以 键值对 形式出现，以 { } 包裹， ， 分割。简单配置示例： 
    $("html").niceScroll({
      touchbehavior: false,//激活拖拽滚动
      cursorcolor: "#E04D00",//滚动条颜色，16进制表示
      cursoropacitymax: 1,//滚动条最大透明度
      cursorwidth: 8,//滚动条宽度
      horizrailenabled: true,//水平滚动
      cursorborderradius: 5,//滚动条圆角
      autohidemode: true,//滚动自动隐藏
      background: 'none',//滚动条背景
      cursorborder: 'none'//滚动条边框
    });
  4. 部分功能：
    • 窗口改变时重置滚动条大小 
      $("#mydiv").getNiceScroll().resize();
    • 滚动至：
      $("#mydiv").getNiceScroll(0).doScrollLeft(x, duration); // X轴至某处
      $("#mydiv").getNiceScroll(0).doScrollTop(y, duration); // Y轴至某处
  5. 使用时遇到的问题：bootstrap 的折叠元素，展开时，滚动条不能重置，导致页面无法滚动到底。
